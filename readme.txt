== Description ==

This plugin allows the admin or moderator to send an email to all group members.
As a member of a group you can view all newsletter

== Installation ==
1. Upload the files to wp-content/plugins/buddypress-group-newsletter
2. Activate the plugin in your admin backend

That's it ... enjoy!

== Upgrade Notice ==

1. Deactivate buddypress-group-newsletter
2. Backup any custom image, stylesheet and language files
3. Upload all files via FTP to wp-content/plugins/buddypress-group-newsletter
4. Restore custom language, stylesheets and image files
5. Activate BuddyPress Introduce
